# graphql_ppx_re

> Experiments with [graphql_ppx](https://github.com/mhallin/graphql_ppx) and Reason

## Developing

```
npm install -g esy@latest
esy install
esy build
```

## Running tests

```
cd tests
yarn install # npm install
yarn test # npm test
```
